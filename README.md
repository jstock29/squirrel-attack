# Squirrel Attack Predictor

My friend hates squirrels. This is for her.


## Install

### Frontend

Requires Node 16

`npm i`

### Backend

Requires Python 3.8

`cd backend`

`virtualenv venv`

`source venv/bin/activate`

`pip install -r requirements-core.txt`

## Run

Frontend

`npm run start` or `npm run web`

API

`uvicorn api:app --reload`
